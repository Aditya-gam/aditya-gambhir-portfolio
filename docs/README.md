## 🛠️ Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript 5
- **Styling**: Tailwind CSS 4.0, CSS Custom Properties
- **UI Components**: Radix UI, shadcn/ui
- **Backend**: Next.js API Routes, MongoDB
- **Deployment**: Vercel
- **Security**: Google reCAPTCHA v2
